# 推荐数量与评分标准说明

## 📊 为什么推荐的股票数量少？

如果您发现系统推荐的股票数量较少（比如只有1-2只），这是**正常现象**，主要有以下几个原因：

### 1. 严格的质量标准

本系统采用**价量时空交易系统**理论，设定了严格的评分标准（默认最低60分），确保推荐的都是高质量的交易机会。

**评分构成**：
- 均线密集度（30分）
- 多头排列（25分）
- 价格突破MA20（15分）
- 成交量放大（10分）
- 盈亏比优势（20分）

只有满足多个条件的股票才能达到60分以上。

### 2. 市场环境影响

推荐数量会随市场环境变化：

- **震荡市场**：大部分股票趋势"未定义"，符合条件的很少
- **单边下跌**：空头排列为主，无买入机会
- **单边上涨**：可能出现"加速上涨"，但位置过高不建议追
- **趋势转折期**：刚形成突破信号的股票较少

### 3. 风险控制理念

**宁缺毋滥**的原则：
- 不符合标准的机会宁可不推荐
- 避免在不确定的市场环境中给出建议
- 保护投资者避免频繁交易和追高

## 🎯 如何调整获得更多推荐？

如果您希望看到更多推荐（接受稍低的质量标准），可以通过以下方式调整：

### 方法1：降低评分阈值

```bash
# 降低到50分（会获得更多推荐）
python run.py --local --min-score 50

# 降低到40分（推荐数量会更多）
python run.py --local --min-score 40

# 在青龙面板中
python ql_task.py --mode full --min-score 50
```

### 方法2：扩大分析范围

```bash
# 分析更多股票（默认100只）
python run.py --local --max-stocks 200

# 在青龙面板中
python ql_task.py --mode full --max-stocks 200 --min-score 55
```

### 方法3：调整配置文件

编辑 `config/config.yaml`:

```yaml
analysis:
  max_stocks: 200        # 增加分析数量
  min_score: 50          # 降低最低分数
  
  # 调整其他过滤条件
  filters:
    min_price: 3.0       # 降低最低价格要求
    min_volume: 500000   # 降低成交量要求
```

## 📈 不同评分阈值的含义

| 评分范围 | 推荐质量 | 适用场景 | 预期数量 |
|---------|---------|---------|---------|
| 80-100分 | ⭐⭐⭐⭐⭐ 极优 | 绝佳机会，强烈推荐 | 很少（0-2只） |
| 70-79分 | ⭐⭐⭐⭐ 优秀 | 高质量机会，推荐买入 | 少量（2-5只） |
| 60-69分 | ⭐⭐⭐ 良好 | 符合标准，可以考虑 | 适中（5-10只） |
| 50-59分 | ⭐⭐ 一般 | 部分条件满足，谨慎 | 较多（10-20只） |
| 40-49分 | ⭐ 观察 | 仅供参考，高风险 | 很多（20+只） |

## 🔍 推荐类型说明

### 1. 密集成交区突破 ⭐⭐⭐

**特征**：
- 均线密集，多头排列刚形成
- 价格突破MA20，成交量放大
- 评分通常在70-100分

**操作**：
- ✅ **买入时机**：突破确认时买入
- 🛡️ **止损设置**：严格按止损位执行
- 💎 **持有管理**：盈利后移动止损

**示例**：
```
【排名 1】 600519 - 密集成交区突破 ⭐⭐⭐
  综合评分: 85分
  入场信号: 突破MA20且均线密集
  盈亏比: 5.2:1
```

### 2. 稳定趋势回调 ⭐⭐⭐

**特征**：
- 已在稳定上升趋势中
- 回踩MA60或MA120支撑
- 评分通常在65-90分

**操作**：
- ✅ **买入时机**：回踩支撑位企稳后
- 🛡️ **止损设置**：跌破支撑位止损
- 💎 **持有管理**：趋势持续则继续持有

**示例**：
```
【排名 2】 000858 - 稳定趋势回调 ⭐⭐⭐
  综合评分: 75分
  入场信号: 回踩MA60支撑
  止损位: 28.50 (-5.2%)
```

### 3. 加速行情-持有 ⭐

**特征**：
- 股价加速上涨，乖离率较大
- 已经涨幅较大
- 评分通常在40-60分

**操作**：
- ⚠️ **不建议新买入**：位置过高，风险大
- 💎 **已有持仓**：可继续持有，设好止盈
- 🚪 **离场信号**：出现顶部构造考虑减仓

**示例**：
```
【排名 5】 300724 - 加速行情-持有 ⭐
  综合评分: 50分
  入场信号: 不建议追高
  持有信号: 已有持仓继续持有
```

## 💡 使用建议

### 对于保守型投资者

建议使用**默认设置**（min_score=60）：
```bash
# 本地运行
python run.py --local --min-score 60 --max-stocks 100

# 青龙面板
python ql_task.py --mode full --min-score 60
```

**特点**：
- 推荐数量少但质量高
- 符合严格的技术标准
- 风险相对可控

### 对于积极型投资者

可以适当降低阈值（min_score=50-55）：
```bash
# 本地运行
python run.py --local --min-score 50 --max-stocks 200

# 青龙面板
python ql_task.py --mode full --min-score 50 --max-stocks 200
```

**特点**：
- 推荐数量增加
- 有更多选择机会
- 需要自己做更多筛选

### 对于学习研究者

可以查看所有分析结果（min_score=0）：
```bash
# 查看所有分析结果（包括不推荐的）
python run.py --local --min-score 0 --max-stocks 300
```

**用途**：
- 了解市场整体状况
- 研究评分机制
- 对比不同评分的标的

## 🎓 深入理解评分体系

### 为什么评分这么严格？

1. **保护资金安全**
   - 避免在不确定的环境中交易
   - 减少因追涨杀跌而亏损

2. **提高胜率**
   - 只在有明确优势时出手
   - 提升交易的成功概率

3. **控制交易频率**
   - 避免过度交易
   - 降低交易成本

### 如何看待"无推荐"的情况？

**"无推荐"本身就是一种信号**：

- ✅ 表示当前市场没有明确机会
- ✅ 建议空仓或轻仓观望
- ✅ 等待更好的入场时机

**记住**：在交易中，"不做"往往比"乱做"更重要！

## ❓ 常见问题

### Q1: 为什么今天只推荐了1只股票？

**A**: 这可能意味着：
- 当前市场环境不适合大量操作
- 大部分股票没有明确的趋势信号
- 系统在保护您避免盲目交易

**建议**：耐心等待，或降低评分阈值（如 --min-score 50）查看更多潜在机会。

### Q2: 如何判断是否应该降低评分阈值？

**A**: 考虑以下因素：
- 您的风险承受能力（高 → 可以降低）
- 您的交易经验（丰富 → 可以自己筛选更多标的）
- 市场环境（活跃 → 可以适当降低）

### Q3: "加速上涨-持有"是买入信号吗？

**A**: **不是**！这是给已持有该股票的投资者的建议。
- 新资金**不建议追高买入**
- 已持有者可以继续持有，但要设好止盈止损
- 出现顶部信号时考虑减仓

### Q4: 如何结合自己的判断使用推荐？

**A**: 系统推荐是辅助工具，建议：
1. 查看Markdown报告，了解推荐理由
2. 自己分析该股票的基本面
3. 结合市场环境做出决策
4. 严格执行风险管理
